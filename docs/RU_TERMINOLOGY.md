# Русская терминология

Этот документ фиксирует правила русского перевода. Канонические
машиночитаемые значения находятся в `data/lang/ru/names.csv` и
`data/lang/ru/glossary.csv`.

## Приоритет источников

1. Источником смысла являются японские выгрузки SCEN и SYSTEM.
2. Существующий английский перевод используется для перекрёстной проверки, но
   не считается безусловно верным.
3. Имена собственные сверяются с принятой терминологией серии и решениями
   проекта.
4. Обычные классы, существа, экипировка, заклинания и военные роли переводятся
   естественными русскими терминами, если смысл японского оригинала однозначен.

Основные внешние источники для сверки:

- <https://www.langrisser.wiki/>
- <https://langrisser.fandom.com/wiki/Langrisser_V>
- <https://gamefaqs.gamespot.com/saturn/562834-langrisser-v-the-end-of-legend/faqs/41339>

Полного устоявшегося русскоязычного справочника по Langrisser V не существует.
Поэтому термины переводятся с японского и проверяются по принятому английскому
канону, а не копируются из несвязанной локализации.

## Покрытие данных

В `names.csv` заполнены 506 исходных ключей:

- 1–170: классы и существа;
- 171–272: оружие, доспехи и аксессуары;
- 273–306: заклинания;
- 307–429: типы войск, роли, титулы и имена на плашках;
- 430–457: состояния и умения;
- 458–506: имена собственные.

В `glossary.csv` дополнительно находятся повторяющиеся термины мира и плашки
говорящих из SCEN, которых нет в таблице имён SYSTEM.

## Принятые решения

- `レインフォルス`: Лейнфорс / Lainforce.
- `ラッセル`: Рассел / Russel.
- `リグリア`: Ригюлер / Rigüler.
- `グロブ`: Гроув / Grove. Титул `死人使い` переводится как «некромант» только
  там, где он семантически присутствует вне короткой плашки имени.
- `クリムゾ`: Кримзо / Crimzo — красная луна.
- `クリムゾニア`: Кримзония / Crimzonia — государство и правящая цивилизация.
- `クリムゾランダー`: ландеры Кримзо / Crimzo Landers — потомки низшего
  сословия. Термин не связан с юнитом `上陸兵` («десант»).
- `クリムゾンナイツ` и `Cナイツ`: Кримзонианские рыцари / Crimzonian
  Knights. Сокращённые формы для интерфейса хранятся только в поле `alt`.
- `ボーゼル`: Бёзер / Böser.
- `エルサリア`: Эль-Саллия / El Sallia.
- `サルラス`: Салрат / Salrath.
- `レイノルズ`: Рейнольдс / Reynolds.
- `ガルツォーク`: Галшок / Galshok.
- `ハディング`: Хаддинг / Hadding.
- `うみにん`: Уминин / Uminin.
- `クノックス`: Нокс / Knox.
- `エストール`: Эстор / Estor.
- `ハーブローク`: Полурух / Half Roc. Это наёмник-летун, а не предмет:
  описание SYSTEM говорит о гигантском орле, «царе неба», сильном против
  других летунов. Вариант `Herb Cloak` считается ошибочной трактовкой
  катаканы.
- `パイク`: Пикинер / Pike. Пеший наёмник типа `槍兵`; описание — воин с
  копьём длиннее трёх метров. «Пика» — название оружия, а не бойца;
  предмета `パイク` в игре нет (таблица предметов `0AF04` его не содержит).
- `槍兵`: пикинеры / Pike (тип бойца). Метка типа в статусе — «Пикинеры»,
  в текстах треугольника типов и в диалогах — «пикинеры», не «копейщики»:
  слово не должно перекликаться с конной линейкой лансьеров.
- `ランサー`: Лансьер / Lancer (также `ヘビーランサー` — Тяжёлый лансьер,
  `ロイヤルランサー` — Королевский лансьер). Это конница (тип `騎兵`,
  описание — «обычный отряд конницы»), поэтому «Копейщик» не годится:
  копейщиками в переводе называется пеший класс `槍兵`, против которого
  конница как раз слаба. «Лансьер» — русское название европейской копейной
  кавалерии, в одном ряду с уже принятым `ドラグーン` — «Драгун»; вариант
  «Улан» отклонён из-за тюркского корня и ассоциации с лёгкой конницей.

Названия заклинаний могут переводиться по подтверждённому эффекту, а не
механически транслитерироваться. Например, `ファイン` переведено как
«Очищение», потому что японские описания называют его средством снятия сна,
смятения, немоты и других состояний. `デクライン` переведено как
«Ослабление», поскольку оно снижает сопротивление магии.

## Плашки говорящих

Титулы не входят в имена на плашках. Поэтому `ギザロフ元帥` и
`死人使いグロブ` в `names.csv` преобразуются в «Гизароф» и «Гроув». Полный титул
переводится в тексте реплики с опорой на японский оригинал и glossary.

Замаплены все 118 уникальных японских ключей из speaker pools. Проверка через
реальный распределитель пар создала 106 уникальных русских плашек; каждая
занимает не более пяти закодированных ячеек, что соответствует ограничению
переносчика строк.

## Проверка

Команда:

```bash
python3 scripts/lang5_validate_terms.py --lang ru --require-complete --require-speakers --max-plate-chars 10
```

Валидатор проверяет:

- полное соответствие набора японских ключей reference-пакету;
- отсутствие пустых обязательных переводов;
- отсутствие разных переводов у одинакового японского ключа;
- непротиворечивость псевдонимов glossary;
- совпадение канонических значений между `names.csv` и `glossary.csv`;
- наличие короткого русского имени для каждой реальной плашки SCEN.
